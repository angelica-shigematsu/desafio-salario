# Desafio Técnico
- Criar um API REST
- Criar cadastro de funcionários
- Alteração dos salários de acordo com o aumento
- Cálculo de imposto

# Rotas
| Rota                        |     Method    |     
| ----------------------------| ------------- | 
| api/adicionar/funcionario   | POST          |
| api/calcular-salario/{cpf}  | PUT           | 
| api/calcular-imposto/{cpf}  | GET           | 

# Tecnologias
- Java
- Spring Boot
- Maven
- JPA
- Hibernate
- Postgres
